## Mission 1: Notion Portal (14-Day Structure)

**Objective**: Create a complete Notion workspace template for the 14-day journey.

**Structure**:
- Welcome page (how to use, what to expect)
- 14 daily pages, each with:
  - Day title + estimated time (Day 1 = 15 min, others = 30-60 min)
  - Today's objective (1 clear sentence)
  - Context/why this matters (2-3 sentences, Older Brother voice)
  - Action steps (numbered, specific)
  - AI Prompt to use (copy-paste ready, formatted in code block)
  - Done checklist (what counts as "complete")
  - Stuck? section (link to OneAI Coach + common fixes)
- Progress tracker (visual, satisfying to check off)
- Resource hub (links to PDFs, GPT, community)
- Launch celebration page (Day 14+)

**Day Breakdown**:
- Day 1: Choose your niche (15 min — quick win, validation framework)
- Day 2: Validate demand (search data, Reddit, competitor check)
- Day 3: Define your product (format, scope, unique angle)
- Day 4: Outline content structure
- Day 5: Write/generate core content (AI-assisted)
- Day 6: Create supplementary materials
- Day 7: Design & formatting (Canva templates)
- Day 8: Set up delivery (Gumroad/platform)
- Day 9: Write sales copy (landing page)
- Day 10: Build social proof & trust elements
- Day 11: Set up marketing funnel (email, socials)
- Day 12: Pre-launch (teaser, audience warm-up)
- Day 13: Launch day execution
- Day 14: Post-launch optimization + first sale push

**Quality**: Clean, minimal, easy to navigate. No clutter. Each day feels achievable.

## Mission 2: Custom GPT — "OneAI Coach"

**Objective**: Build the system prompt and knowledge base for a Custom GPT on ChatGPT.

**System Prompt Core Behaviors**:
1. Always knows what day the user is on (asks if unclear)
2. Gives ONE next step, never overwhelms
3. Celebrates small wins genuinely
4. Detects "stuck" patterns (vague questions, repeated topics, no progress mentions)
5. When stuck detected: acknowledge → reframe → give micro-action (5 min max)
6. Never suggests buying other products
7. Uses "Experienced Older Brother" voice consistently
8. Keeps responses under 200 words unless user asks for detail
9. References the Notion portal and PDFs when relevant
10. Asks follow-up questions to keep momentum

**Knowledge Files to Upload**:
- Master Guide PDF content
- 50 Prompts Library
- Common Mistakes Guide
- Day-by-day objectives summary
- FAQ/objection handling

**Conversation Starters**:
- "I just bought One Product AI. What do I do first?"
- "I'm on Day [X] and I'm stuck."
- "Help me validate my niche idea."
- "I'm not sure my product is good enough to sell."

**Boundaries**:
- Won't give legal/tax advice (redirects to professional)
- Won't build the product FOR them (guides, doesn't do)
- Won't engage with topics outside digital product creation
- If user hasn't started: gently pushes to Day 1, no judgment

## Mission 5: Pre-Purchase Chatbot

**Objective**: Build a chat widget that answers visitor questions and guides toward purchase.

**Tech Stack**:
- Frontend: Floating chat bubble (bottom-right), opens mini chat window
- Backend: OpenAI API (GPT-4o-mini for cost efficiency)
- Hosted: Vercel serverless function or similar

**Chatbot Personality**:
- Same "Older Brother" voice but slightly more sales-aware
- Honest about what the product is and isn't
- Uses real stats when answering objections
- Never pushy, never fake-urgent
- If someone clearly isn't a fit, says so honestly

**Knowledge It Should Have**:
- Full product details (what's included, pricing, guarantee)
- Market stats (8.4M side hustlers, 62% quit rate, etc.)
- FAQ answers
- Competitor comparisons (honest, not trash-talking)
- Founder story (briefly)

**Conversation Flows**:
1. Greeting → What brings you here?
2. Objection handling (too cheap? scam? will it work for me?)
3. Product explanation (what you get, how it works)
4. Pricing/guarantee clarification
5. Ready to buy → link to checkout
6. Not ready → offer email signup for free resource

**Constraints**:
- Max 150 words per response
- No hallucinating features that don't exist
- Must disclose it's an AI assistant if asked
- Rate limited (prevent abuse)
- Conversation stored for product improvement (with consent)
